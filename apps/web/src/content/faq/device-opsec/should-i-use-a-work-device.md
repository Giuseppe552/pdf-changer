# Should I use a work device for sensitive documents?

Usually no. Work-managed devices can be monitored, logged, backed up, and audited by your organization.

## Why this matters

Even if the file is scrubbed correctly, endpoint monitoring can expose when and how the file was handled.

## Safe default steps

1. Avoid employer-managed devices for high-risk tasks.
2. Separate sensitive activity from normal personal/work accounts.
3. Minimize installed software and background sync during processing.

## Common mistakes

- Believing browser privacy mode bypasses endpoint monitoring.
- Assuming "my company would never check logs."
- Mixing sensitive files with synced workplace folders.

## Limits

No single device setup is universally safe. Choose a workflow based on your actual risk and legal context.
