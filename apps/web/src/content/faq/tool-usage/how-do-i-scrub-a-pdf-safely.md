# How do I scrub a PDF safely?

Use a short repeatable workflow: scrub locally, read the report, inspect the output, and only share the sanitized copy.

## Why this matters

Most failures are process errors, not software bugs. A calm checklist reduces accidental leaks.

## Safe default steps

1. Open [Deep Metadata Scrubber](/scrub).
2. Drag in the file and run scrub.
3. Read the scrub report and hashes.
4. Open the output and inspect all pages.
5. Share only the scrubbed copy.

## Common mistakes

- Sending the original by mistake.
- Skipping output review because of urgency.
- Forgetting visible identifiers in the content.

## Limits

Scrubbing reduces metadata risk but cannot guarantee anonymity against strong adversaries.
