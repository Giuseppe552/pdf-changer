# Can I use PDF Changer offline?

Yes, after first load. The PWA can run scrubber flows offline within local usage limits.

## Why this matters

Offline processing can reduce network exposure while you handle sensitive files.

## Safe default steps

1. Visit the app once while online.
2. Confirm the page loads correctly.
3. Disconnect network access.
4. Process your file locally.
5. Reconnect only when you need to transmit output.

## Common mistakes

- Assuming offline mode fixes a compromised device.
- Forgetting cached data can be cleared by browser cleanup.
- Trying account upgrades while offline.

## Limits

Offline mode helps with network exposure, not device malware or visible content identifiers.
